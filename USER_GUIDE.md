## 1. Pendahuluan
Aplikasi Trial Validation System digunakan untuk mengelola proses validasi trial produk, mulai dari input data, validasi, review, hingga approval. Setiap user memiliki hak akses dan tugas berbeda.

---

## 2. Panduan untuk Staff
### A. Login
- Masukkan email dan password pada halaman login.

### B. Membuat Trial Baru
1. Klik menu **Trial** atau **Dashboard**.
2. Pilih **Buat Trial Baru**.
3. Isi data produk, scope validasi, mesin, dan data lain yang diminta.
4. Klik **Simpan**.

### C. Input Data Validasi
1. Setelah trial dibuat, lanjut ke halaman **Validation**.
2. Isi parameter validasi sesuai instruksi.
3. Klik **Simpan**.

### D. Input Data Weighing
1. Masuk ke menu **Weighing**.
2. Input data penimbangan (Packaging & Filling).
3. Klik **Simpan**.

### E. Upload Lampiran
1. Masuk ke menu **Attachments**.
2. Pilih kategori dan upload file gambar.
3. Klik **Simpan**.

### F. Submit Trial untuk Review
1. Setelah semua data lengkap, klik **Submit for Review**.
2. Pilih departemen reviewer yang dituju.
3. Konfirmasi pengiriman.

### G. Revisi Trial
- Jika trial dikembalikan (Need Revision), lakukan perbaikan lalu submit ulang.

---

## 3. Panduan untuk Reviewer
### A. Login
- Masukkan email dan password pada halaman login.

### B. Melihat Daftar Trial untuk Review
1. Klik menu **Reviews**.
2. Lihat daftar trial yang membutuhkan review departemen Anda.

### C. Melakukan Review
1. Klik trial yang ingin direview.
2. Baca data trial, validasi, dan lampiran.
3. Isi komentar review.
4. Klik **Submit Review**.

---

## 4. Panduan untuk Manager QAC
### A. Login
- Masukkan email dan password pada halaman login.

### B. Approval Trial
1. Klik menu **Approvals**.
2. Lihat daftar trial yang siap di-approve.
3. Klik trial, baca data dan review.
4. Pilih **Approve** atau **Reject**.
5. Isi komentar dan password e-signature.
6. Klik **Submit**.

---

## 5. FAQ & Bantuan
- Jika lupa password, hubungi admin.
- Jika ada error atau kendala, catat pesan error dan hubungi admin/support.
- Gunakan menu **Notifications** untuk melihat notifikasi terbaru.
